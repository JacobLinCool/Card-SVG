card-svg

Card SVG

Create SVG Cards.

Example

 
import { Card } from "card-svg";

const card = new Card()
    .size(400, 200)
    .theme("nord")
    .background()
    .text("JacobLinCool", "text1", { x: 20, y: 30 });

const svg = card.export();

  • height: number
